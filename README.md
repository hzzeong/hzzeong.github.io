## 실행 방법

이 프로젝트를 로컬에서 실행하려면 다음 단계를 따르세요:

1. 리포지토리를 클론합니다:
    ```bash
    git clone https://github.com/hzzeong/hzzeong.github.io.git
    ```

2. 프로젝트 디렉토리로 이동합니다:
    ```bash
    cd your-repo-name
    ```

3. 선호하는 웹 브라우저에서 `index.html`을 엽니다.
